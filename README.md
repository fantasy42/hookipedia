**Hookipedia** is a comprehensive documentation website for discovering, evaluating, and understanding high-quality custom React hooks. Think of it as Wikipedia for the React hooks ecosystem.

## Core Purpose

React developers often struggle to:

- Find reliable, well-tested custom hooks
- Evaluate hook quality and edge case handling
- Understand implementation details and trade-offs
- Discover alternative hooks for the same problem

Hookipedia solves these challenges by providing structured, community-vetted documentation that helps developers make informed decisions about which hooks to use in their projects.

## Features

- **Comprehensive Documentation**: Detailed explanations of hooks including use cases, API reference, and implementation insights
- **Implementation Variants**: Multiple implementation approaches for common patterns
- **Use Case Guides**: Practical examples showing hooks in real-world scenarios
- **Source Analysis**: Links to source code with inline annotations and explanations

## Quick Start

### Prerequisites

- Node.js 20+
- Bun

### Local Development

1. **Clone the repository**

```bash
git clone https://github.com/fantasy42/hookipedia.git
cd hookipedia
```

2. Install dependencies (using Bun)

```bash
bun install
```

3. Start the development server

```bash
bun dev
```

4. Open your browser

Navigate to <http://localhost:3000>

## How to Contribute

We welcome contributions from the React community! Here are ways you can help:

1. **Improving Docs**: Fix typos or clarify explanations, add more examples or use cases, translate documentation to other languages.

2. **Submitting a Hook**: Share a pattern you've perfected.

3. **Report Issues**: Found a bug? [Open an issue](https://github.com/fantasy42/hookipedia/issues).
